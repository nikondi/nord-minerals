## Пагинация новостей и проектов
[[Обратно]](../README.md)
### Общая информация

Список новостей находится в шаблоне `pages/news/index.html.twig`, элемент `.news-list-wrapper`;  
список проектов в `pages/projects.html.twig`, элемент `.projects-list-wrapper`

**Настройка списков**:

Endpoint для пагинации указывается в аттрибуте враппера `data-endpoint`, кол-во
подгружаемых элементов - в аттрибуте `data-per_page`
